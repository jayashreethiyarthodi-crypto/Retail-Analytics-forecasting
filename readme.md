## Overview
This project analyzes retail sales data to identify trends, top-performing
products, and revenue distribution across stores and categories.

## Tools Used
- Python (Pandas)
- Jupyter Notebook
- Tableau (Treemap Visualizations)

## Key Insights
- Identified top 3 products by net sales
- Analyzed category and store-level performance
- Supported pricing and inventory decisions through data storytelling
## Forecasting
A linear regression model was used to forecast future retail demand based on
historical monthly net sales. The model projects short-term sales trends to
support inventory planning and pricing decisions.
